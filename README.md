# Plain Web

This is web ui source code of [Plain App](https://github.com/ismartcoding/plain-app).

## Run

Duplicate `.env` file to `.env.local` and update the `VITE_APP_API_HOST`.

```sh
yarn
yarn dev
```

## Build

```sh
yarn build
```

### Lint

```sh
yarn lint
```
